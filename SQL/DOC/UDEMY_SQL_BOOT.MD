<h3 id='21'>Restoring a Table Schema</h3>

To the new DB

```
newdb >>> restore >>> find the .tar file >>> restore options >>> only schema YES >>> restore >>> refresh
```

To the old DB

```
olddb >>> restore >>> find the .tar file >>> restore options >>> only schema YES >>> clean before restore YES 
>>> restore >>> refresh
```

<h3 id='37'>IN</h3>

```SQL
SELECT customer_id, rental_id, return_date
FROM rental
WHERE customer_id IN (1,2)
ORDER BY return_date DESC;
```

```SQL
SELECT customer_id, rental_id, return_date
FROM rental
WHERE customer_id IN (1,2,7,21)
ORDER BY return_date DESC;
```

```SQL
SELECT customer_id, rental_id, return_date
FROM rental
WHERE customer_id NOT IN (1,2)
ORDER BY return_date DESC;
```

<h3 id='40'>MIN MAX SUM and AVG</h3>

AVG

```SQL
SELECT AVG(amount)
FROM payment;
```	

ROUND

```SQL
--tizedes vesszo utan 2 szamjegyet ad vissza
SELECT ROUND (AVG(amount), 2)
FROM payment;
```

MIN

```SQL
SELECT MIN(amount)
FROM payment;
```

MAX

```SQL
SELECT MAX(amount)
FROM payment;
```

SUM

```
SELECT SUM(amount)
FROM payment;
```

<h3 id='41'>GROUP BY</h3>

```SQL
SELECT customer_id, SUM(amount)
FROM payment
GROUP BY customer_id;
```

```SQL
SELECT customer_id, SUM(amount)
FROM payment
GROUP BY customer_id
ORDER BY SUM(amount) ASC;
```

```SQL
SELECT staff_id, COUNT(payment_id)
FROM payment
GROUP BY staff_id;
```

```SQL
SELECT store_id, COUNT(customer_id)
FROM customer
GROUP BY store_id;
```
